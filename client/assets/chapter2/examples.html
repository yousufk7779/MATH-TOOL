Example 2. Find the zeroes of the quadratic polynomial x² + 7x + 10, and verify the relationship between the zeroes and the coefficients.
Solution:
Let p(x) = x² + 7x + 10
To find zeroes, put p(x) = 0
By splitting the middle term:
x² + 2x + 5x + 10 = 0
x(x + 2) + 5(x + 2) = 0
(x + 2)(x + 5) = 0
x + 2 = 0 OR x + 5 = 0
x = -2 OR x = -5
Zeroes are α = -2 and β = -5.
Verification:
Comparing with ax² + bx + c: a = 1, b = 7, c = 10
Sum of zeroes:
α + β = -2 + (-5) = -7
-b/a = -7/1 = -7
Matches.
Product of zeroes:
α × β = (-2) × (-5) = 10
c/a = 10/1 = 10
Matches.
Relationship verified.
Example 3. Find the zeroes of the polynomial x² - 3 and verify the relationship between the zeroes and the coefficients.
Solution:
Let p(x) = x² - 3
To find zeroes, put p(x) = 0
x² - 3 = 0
x² - (√3)² = 0
Using identity a² - b² = (a - b)(a + b)
(x - √3)(x + √3) = 0
x - √3 = 0 OR x + √3 = 0
x = √3 OR x = -√3
Zeroes are α = √3 and β = -√3.
Verification:
Comparing with ax² + bx + c: a = 1, b = 0 (no x term), c = -3
Sum of zeroes:
α + β = √3 + (-√3) = 0
-b/a = -0/1 = 0
Product of zeroes:
α × β = √3 × (-√3) = -3
c/a = -3/1 = -3
Relationship verified.
Example 4. Find a quadratic polynomial, the sum and product of whose zeroes are -3 and 2, respectively.
Solution:
Given:
Sum of zeroes (S) = -3
Product of zeroes (P) = 2
A quadratic polynomial is given by:
p(x) = k[x² - (Sum of zeroes)x + (Product of zeroes)]
p(x) = k[x² - (-3)x + 2]
p(x) = k[x² + 3x + 2]
For k = 1, the polynomial is:
x² + 3x + 2
Example 5. Verify that 3, -1, -⅓ are the zeroes of the cubic polynomial p(x) = 3x³ - 5x² - 11x - 3, and then verify the relationship between the zeroes and the coefficients.
Solution:
Given polynomial: p(x) = 3x³ - 5x² - 11x - 3
Checking 3:
p(3) = 3(3)³ - 5(3)² - 11(3) - 3
= 3(27) - 5(9) - 33 - 3
= 81 - 45 - 36
= 81 - 81 = 0
Checking -1:
p(-1) = 3(-1)³ - 5(-1)² - 11(-1) - 3
= -3 - 5 + 11 - 3
= -11 + 11 = 0
Checking -1/3:
p(-1/3) = 3(-1/3)³ - 5(-1/3)² - 11(-1/3) - 3
= 3(-1/27) - 5(1/9) + 11/3 - 3
= -1/9 - 5/9 + 33/9 - 27/9
= (-1 - 5 + 33 - 27) / 9
= 0 / 9 = 0
Thus, 3, -1, and -1/3 are indeed zeroes of p(x).
Verification of Relationship:
Comparing with ax³ + bx² + cx + d:
a = 3, b = -5, c = -11, d = -3
Let α = 3, β = -1, γ = -1/3
1. Sum of zeroes (α + β + γ):
3 + (-1) + (-1/3) = 2 - 1/3 = 53
-b/a = -(-5)/3 = 53
2. Sum of product of zeroes taken two at a time (αβ + βγ + γα):
(3)(-1) + (-1)(-1/3) + (-1/3)(3)
= -3 + 1/3 - 1
= -4 + 1/3 = -113
c/a = -113
3. Product of zeroes (αβγ):
(3) × (-1) × (-1/3) = 1
-d/a = -(-3)/3 = 3/3 = 1
Relationship verified.
