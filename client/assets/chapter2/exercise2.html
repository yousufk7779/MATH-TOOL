1. Find the zeroes of the following quadratic polynomials and verify the relationship between the zeroes and the coefficients.
(i) x² - 2x - 8
Solution:
Split middle term -2x as -4x + 2x:
x² - 4x + 2x - 8 = 0
x(x - 4) + 2(x - 4) = 0
(x - 4)(x + 2) = 0
So, x = 4 or x = -2
Zeroes are 4 and -2.
Verification:
Here a=1, b=-2, c=-8
Sum: 4 + (-2) = 2. Formula: -b/a = -(-2)/1 = 2. (Verified)
Product: 4 × (-2) = -8. Formula: c/a = -8/1 = -8. (Verified)
(ii) 4s² - 4s + 1
Solution:
Split -4s as -2s - 2s:
4s² - 2s - 2s + 1 = 0
2s(2s - 1) - 1(2s - 1) = 0
(2s - 1)(2s - 1) = 0
Zeroes are 1/2 and 1/2.
Verification:
Here a=4, b=-4, c=1
Sum: 1/2 + 1/2 = 1. Formula: -b/a = -(-4)/4 = 1. (Verified)
Product: 1/2 × 1/2 = 1/4. Formula: c/a = 1/4. (Verified)
(iii) 6x² - 3 - 7x
Solution:
Rearrange: 6x² - 7x - 3 = 0
Split -7x as -9x + 2x:
6x² - 9x + 2x - 3 = 0
3x(2x - 3) + 1(2x - 3) = 0
(2x - 3)(3x + 1) = 0
Zeroes are 3/2 and -1/3.
Verification:
Here a=6, b=-7, c=-3
Sum: 3/2 + (-1/3) = 7/6. Formula: -b/a = -(-7)/6 = 7/6. (Verified)
Product: 3/2 × (-1/3) = -3/6 = -1/2. Formula: c/a = -3/6 = -1/2. (Verified)
(iv) 4u² + 8u
Solution:
Take 4u common:
4u(u + 2) = 0
4u = 0 ⇒ u = 0
u + 2 = 0 ⇒ u = -2
Zeroes are 0 and -2.
Verification:
Here a=4, b=8, c=0
Sum: 0 + (-2) = -2. Formula: -b/a = -8/4 = -2. (Verified)
Product: 0 × (-2) = 0. Formula: c/a = 0/4 = 0. (Verified)
(v) t² - 15
Solution:
Use a² - b² = (a-b)(a+b):
t² - (√15)² = 0
(t - √15)(t + √15) = 0
Zeroes are √15 and -√15.
Verification:
Here a=1, b=0, c=-15
Sum: √15 - √15 = 0. Formula: -b/a = 0. (Verified)
Product: √15 × (-√15) = -15. Formula: c/a = -15/1 = -15. (Verified)
(vi) 3x² - x - 4
Solution:
Split -x as -4x + 3x:
3x² - 4x + 3x - 4 = 0
x(3x - 4) + 1(3x - 4) = 0
(3x - 4)(x + 1) = 0
Zeroes are 4/3 and -1.
Verification:
Here a=3, b=-1, c=-4
Sum: 4/3 + (-1) = 1/3. Formula: -b/a = -(-1)/3 = 1/3. (Verified)
Product: 4/3 × (-1) = -4/3. Formula: c/a = -4/3. (Verified)
