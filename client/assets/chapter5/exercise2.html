Q1. Fill in the blanks. Formula used: aₙ = a + (n - 1)d
(i) a = 7, d = 3, n = 8. Find aₙ.
aₙ = 7 + (8 - 1)3 aₙ = 7 + (7)3 aₙ = 7 + 21 aₙ = 28 Answer: 28
(ii) a = -18, n = 10, aₙ = 0. Find d.
0 = -18 + (10 - 1)d 18 = 9d d = 18 ÷ 9 d = 2 Answer: d = 2
(iii) d = -3, n = 18, aₙ = -5. Find a.
-5 = a + (18 - 1)(-3) -5 = a + (17)(-3) -5 = a - 51 a = -5 + 51 a = 46 Answer: a = 46
(iv) a = -18.9, d = 2.5, aₙ = 3.6. Find n.
3.6 = -18.9 + (n - 1)2.5 3.6 + 18.9 = (n - 1)2.5 22.5 = (n - 1)2.5 n - 1 = 22.5 ÷ 2.5 n - 1 = 9 n = 9 + 1 = 10 Answer: n = 10
(v) a = 3.5, d = 0, n = 105. Find aₙ.
aₙ = 3.5 + (105 - 1)0 aₙ = 3.5 + 0 aₙ = 3.5 Answer: 3.5
Q2. Choose correct choice.
(i) 30th term of AP: 10, 7, 4...
a = 10, d = 7 - 10 = -3, n = 30 a₃₀ = 10 + (30 - 1)(-3) a₃₀ = 10 + 29(-3) a₃₀ = 10 - 87 a₃₀ = -77 Answer: (C) -77
(ii) 11th term of AP: -3, -12, 2...
a = -3, n = 11 d = -12 - (-3) d = -12 + 3 = 52 a₁₁ = -3 + (11 - 1)52 a₁₁ = -3 + 10 × 52 a₁₁ = -3 + 25 a₁₁ = 22 Answer: (B) 22
Q3. Find missing terms in boxes.
(i) 2, □, 26
a = 2, a₃ = 26 a + 2d = 26 2 + 2d = 26 2d = 24 d = 12 Missing term = a + d = 2 + 12 = 14 Answer: 14
(ii) □, 13, □, 3
a₂ = a + d = 13 a₄ = a + 3d = 3 Subtracting eq(1) from eq(2): 2d = -10 d = -5 a = 13 - d = 13 - (-5) = 18 a₃ = a + 2d = 18 + 2(-5) = 8 Answer: 18, 8
(iii) 5, □, □, 912
a = 5, a₄ = 9.5 = 192 a + 3d = 192 5 + 3d = 192 3d = 192 - 5 = 92 d = 32 a₂ = 5 + 32 = 132 = 612 a₃ = 132 + 32 = 162 = 8 Answer: 612, 8
(iv) -4, □, □, □, □, 6
a = -4, a₆ = 6 a + 5d = 6 -4 + 5d = 6 5d = 10 → d = 2 Terms: -2, 0, 2, 4 Answer: -2, 0, 2, 4
(v) □, 38, □, □, □, -22
a₂ = a + d = 38 a₆ = a + 5d = -22 Subtracting: 4d = -60 → d = -15 a = 38 - (-15) = 53 a₃ = 38 - 15 = 23 a₄ = 23 - 15 = 8 a₅ = 8 - 15 = -7 Answer: 53, 23, 8, -7
Q4. Which term of the AP: 3, 8, 13, 18... is 78?
a = 3, d = 8 - 3 = 5, aₙ = 78 78 = 3 + (n - 1)5 75 = (n - 1)5 15 = n - 1 n = 16 Answer: 16th term
Q5. Find number of terms.

a = 7, d = 6, aₙ = 205 205 = 7 + (n - 1)6 198 = (n - 1)6 33 = n - 1 n = 34 Answer: 34 terms
(ii) 18, 1512, 13, ..., -47

Q6. Check if -150 is a term of 11, 8, 5, 2...
a = 11, d = -3 -150 = 11 + (n - 1)(-3) -161 = (n - 1)(-3) n - 1 = 161/3 161 is not divisible by 3. So n is not an integer. Answer: No
Q7. Find 31st term if 11th term is 38 and 16th term is 73.
a₁₁ = a + 10d = 38 a₁₆ = a + 15d = 73 Subtracting: 5d = 35 → d = 7 a + 10(7) = 38 a = 38 - 70 = -32 a₃₁ = a + 30d a₃₁ = -32 + 30(7) a₃₁ = -32 + 210 = 178 Answer: 178
Q8. AP of 50 terms. 3rd term = 12, Last term = 106. Find 29th term.
Total terms n = 50. So a₅₀ = 106. a₃ = a + 2d = 12 a₅₀ = a + 49d = 106 Subtracting: 47d = 94 → d = 2 a + 2(2) = 12 → a = 8 a₂₉ = a + 28d a₂₉ = 8 + 28(2) a₂₉ = 8 + 56 = 64 Answer: 64
Q9. 3rd term is 4, 9th term is -8. Which term is 0?
a₃ = a + 2d = 4 a₉ = a + 8d = -8 Subtracting (2)-(1): 6d = -12 d = -2 a + 2(-2) = 4 → a - 4 = 4 → a = 8 Let nth term be 0. 0 = 8 + (n - 1)(-2) -8 = (n - 1)(-2) 4 = n - 1 n = 5 Answer: 5th term
Q10. 17th term exceeds 10th term by 7. Find common difference.
a₁₇ - a₁₀ = 7 (a + 16d) - (a + 9d) = 7 a + 16d - a - 9d = 7 7d = 7 d = 1 Answer: 1
Q11. Which term of 3, 15, 27, 39... will be 132 more than 54th term?
a = 3, d = 12 a₅₄ = 3 + 53(12) a₅₄ = 3 + 636 = 639 Required term = 639 + 132 = 771 771 = 3 + (n - 1)12 768 = (n - 1)12 64 = n - 1 n = 65 Answer: 65th term
Q12. Two APs have same common difference. Diff between 100th terms is 100. Diff between 1000th terms?
Let APs be a, a+d... and A, A+d... Diff between nth terms = (A + (n-1)d) - (a + (n-1)d) = A - a (constant difference) Given diff between 100th terms = 100. So A - a = 100. Diff between 1000th terms will also be A - a. So difference is 100. Answer: 100
Q13. How many 3-digit numbers are divisible by 7?
3-digit numbers: 100 to 999. First divisible by 7: 105 (7 × 15) Last divisible by 7: 994 (7 × 142) AP: 105, 112, ..., 994 a = 105, d = 7, aₙ = 994 994 = 105 + (n - 1)7 889 = (n - 1)7 127 = n - 1 n = 128 Answer: 128
Q14. How many multiples of 4 lie between 10 and 250?
First multiple > 10 is 12. Last multiple < 250 is 248. AP: 12, 16, ..., 248 a = 12, d = 4, aₙ = 248 248 = 12 + (n - 1)4 236 = (n - 1)4 59 = n - 1 n = 60 Answer: 60
Q15. For what value of n, terms of 63, 65, 67... and 3, 10, 17... are equal?
AP1: a = 63, d = 2 → nth term = 63 + (n-1)2 AP2: a = 3, d = 7 → nth term = 3 + (n-1)7 Equal: 63 + 2n - 2 = 3 + 7n - 7 61 + 2n = 7n - 4 65 = 5n n = 13 Answer: 13
Q16. Determine AP: 3rd term is 16, 7th term exceeds 5th by 12.
a₃ = 16 a₇ - a₅ = 12 (a + 6d) - (a + 4d) = 12 2d = 12 → d = 6 a₃ = a + 2d = 16 a + 2(6) = 16 a + 12 = 16 → a = 4 AP: 4, 10, 16, 22... Answer: 4, 10, 16, 22...
Q17. Find the 20th term from the last term of the AP: 3, 8, 13, ..., 253.
For last term, we reverse the AP. New AP: 253, ..., 13, 8, 3 a = 253, d = 3 - 8 = -5 Find 20th term (a₂₀). a₂₀ = a + 19d a₂₀ = 253 + 19(-5) a₂₀ = 253 - 95 a₂₀ = 158 Answer: 158
Q18. Sum of 4th and 8th terms is 24. Sum of 6th and 10th terms is 44. Find AP.
a₄ + a₈ = 24 (a + 3d) + (a + 7d) = 24 2a + 10d = 24 → a + 5d = 12 ...(1) a₆ + a₁₀ = 44 (a + 5d) + (a + 9d) = 44 2a + 14d = 44 → a + 7d = 22 ...(2) Subtracting (1) from (2): 2d = 10 → d = 5 From (1): a + 5(5) = 12 a + 25 = 12 → a = -13 Terms: -13, -8, -3... Answer: -13, -8, -3...
Q19. Subba Rao started at ₹ 5000 (1995), increment ₹ 200. When ₹ 7000?
a = 5000, d = 200, aₙ = 7000 7000 = 5000 + (n - 1)200 2000 = (n - 1)200 10 = n - 1 n = 11 Year = 1995 + 10 = 2005 (11th year) Answer: In 11th year (2005)
Q20. Ramkali saved ₹ 5 first week, increased by ₹ 1.75. nth week ₹ 20.75. Find n.
a = 5, d = 1.75, aₙ = 20.75 20.75 = 5 + (n - 1)1.75 15.75 = (n - 1)1.75 n - 1 = 15.75 ÷ 1.75 n - 1 = 1575 ÷ 175 n - 1 = 9 n = 10 Answer: n = 10
