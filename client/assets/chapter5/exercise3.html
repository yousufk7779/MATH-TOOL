Exercise 5.3

Sum of n terms: Sₙ = n2[2a + (n - 1)d] Or: Sₙ = n2(a + aₙ)
Q1. Find the sum of the following APs:
(i) 2, 7, 12..., to 10 terms.
a = 2, d = 7 - 2 = 5, n = 10 S₁₀ = 102[2(2) + (10 - 1)5] S₁₀ = 5[4 + 9(5)] S₁₀ = 5[4 + 45] S₁₀ = 5(49) S₁₀ = 245 Answer: 245
(ii) -37, -33, -29..., to 12 terms.
a = -37, d = -33 - (-37) = 4, n = 12 S₁₂ = 122[2(-37) + (12 - 1)4] S₁₂ = 6[-74 + 11(4)] S₁₂ = 6[-74 + 44] S₁₂ = 6(-30) S₁₂ = -180 Answer: -180
(iii) 0.6, 1.7, 2.8..., to 100 terms.
a = 0.6, d = 1.1, n = 100 S₁₀₀ = 1002[2(0.6) + (100 - 1)1.1] S₁₀₀ = 50[1.2 + 99(1.1)] S₁₀₀ = 50[1.2 + 108.9] S₁₀₀ = 50(110.1) S₁₀₀ = 5505 Answer: 5505
(iv) 115, 112, 110..., to 11 terms.
a = 115 d = 112 - 115 = 5 - 460 = 160 S₁₁ = 112[2(115) + (11 - 1)160] S₁₁ = 112[215 + 1060] S₁₁ = 112[215 + 16] S₁₁ = 112[4 + 530] S₁₁ = 112[930] = 3320 Answer: 3320

Q3. In an AP:
(i) a = 5, d = 3, aₙ = 50. Find n and Sₙ.
50 = 5 + (n - 1)3 45 = (n - 1)3 15 = n - 1 → n = 16 S₁₆ = 162(5 + 50) S₁₆ = 8(55) = 440 Answer: n = 16, Sₙ = 440
(ii) a = 7, a₁₃ = 35. Find d and S₁₃.
a₁₃ = a + 12d = 35 7 + 12d = 35 12d = 28 d = 2812 = 73 S₁₃ = 132(7 + 35) S₁₃ = 132(42) = 13(21) = 273 Answer: d = 73, S₁₃ = 273
(iii) a₁₂ = 37, d = 3. Find a and S₁₂.
a + 11d = 37 a + 11(3) = 37 a + 33 = 37 → a = 4 S₁₂ = 122(4 + 37) S₁₂ = 6(41) = 246 Answer: a = 4, S₁₂ = 246
(iv) a₃ = 15, S₁₀ = 125. Find d and a₁₀.
a + 2d = 15 ...(1) S₁₀ = 102[2a + 9d] = 125 5[2a + 9d] = 125 2a + 9d = 25 ...(2) From (1), 2a + 4d = 30 ...(Multiply by 2) Subtracting: 5d = -5 → d = -1 a = 15 - 2(-1) = 17 a₁₀ = a + 9d = 17 + 9(-1) = 8 Answer: d = -1, a₁₀ = 8
(v) d = 5, S₉ = 75. Find a and a₉.
S₉ = 92[2a + 8(5)] = 75 92[2a + 40] = 75 9(a + 20) = 75 9a + 180 = 75 9a = -105 a = -353 a₉ = a + 8d = -353 + 40 a₉ = -35 + 1203 = 853 Answer: a = -353, a₉ = 853
(vi) a = 2, d = 8, Sₙ = 90. Find n and aₙ.
90 = n2[2(2) + (n - 1)8] 180 = n[4 + 8n - 8] 180 = n[8n - 4] 180 = 8n² - 4n 2n² - n - 45 = 0 2n² - 10n + 9n - 45 = 0 2n(n - 5) + 9(n - 5) = 0 (n - 5)(2n + 9) = 0 n = 5 a₅ = 2 + 4(8) = 34 Answer: n = 5, aₙ = 34
(vii) a = 8, aₙ = 62, Sₙ = 210. Find n and d.
Sₙ = n2(a + aₙ) 210 = n2(8 + 62) 210 = n2(70) 210 = 35n → n = 6 a₆ = a + 5d = 62 8 + 5d = 62 5d = 54 → d = 545 Answer: n = 6, d = 545
(viii) aₙ = 4, d = 2, Sₙ = -14. Find n and a.
a + (n - 1)2 = 4 a + 2n - 2 = 4 → a = 6 - 2n ...(1) Sₙ = n2(a + aₙ) = -14 n(a + 4) = -28 Substitute (1): n(6 - 2n + 4) = -28 n(10 - 2n) = -28 10n - 2n² = -28 2n² - 10n - 28 = 0 n² - 5n - 14 = 0 (n - 7)(n + 2) = 0 n = 7 a = 6 - 2(7) = -8 Answer: n = 7, a = -8
(ix) a = 3, n = 8, S = 192. Find d.
192 = 82[2(3) + 7d] 192 = 4[6 + 7d] 48 = 6 + 7d 7d = 42 → d = 6 Answer: d = 6
(x) l = 28, S = 144, n = 9. Find a.
S = n2(a + l) 144 = 92(a + 28) 32 = a + 28 a = 4 Answer: a = 4

a = 9, d = 8, Sₙ = 636 636 = n2[18 + (n - 1)8] 1272 = n[18 + 8n - 8] 1272 = n[10 + 8n] 8n² + 10n - 1272 = 0 4n² + 5n - 636 = 0 b² - 4ac = 25 - 4(4)(-636) = 25 + 10176 = 10201 √D = 101 n = -5 ± 1018 n = 12 (positive) Answer: 12 terms

Q9. Sum of 7 terms is 49, sum of 17 terms is 289. Find sum of n terms.
S₇ = 49 → 72[2a+6d] = 49 7(a+3d) = 49 → a + 3d = 7 ...(1) S₁₇ = 289 → 172[2a+16d] = 289 17(a+8d) = 289 → a + 8d = 17 ...(2) Subtracting (1) from (2): 5d = 10 → d = 2 a + 6 = 7 → a = 1 Sₙ = n2[2(1) + (n-1)2] Sₙ = n2[2 + 2n - 2] Sₙ = n2(2n) = n² Answer: n²

Q11. Sum of first n terms is 4n - n². Find S₁, S₂, 2nd term, 3rd, 10th, nth terms.
Sₙ = 4n - n² S₁ = 4(1) - 1² = 3 (First term a is S₁) S₂ = 4(2) - 2² = 8 - 4 = 4 a₂ = S₂ - S₁ = 4 - 3 = 1 a₃ = S₃ - S₂ S₃ = 4(3) - 3² = 12 - 9 = 3 a₃ = 3 - 4 = -1 a₁₀ = S₁₀ - S₉ S₁₀ = 40 - 100 = -60 S₉ = 36 - 81 = -45 a₁₀ = -60 - (-45) = -15 aₙ = Sₙ - Sₙ₋₁ aₙ = (4n - n²) - [4(n - 1) - (n - 1)²] aₙ = 4n - n² - [4n - 4 - (n² - 2n + 1)] aₙ = 4n - n² - 4n + 4 + n² - 2n + 1 aₙ = 5 - 2n Answer: S₁=3, S₂=4, a₂=1, a₃=-1, a₁₀=-15, aₙ=5-2n

Q18. Spiral of semicircles. Radii 0.5, 1.0, 1.5... Total length of 13 spirals?
Perimeter of semicircle = πr l₁ = π(0.5), l₂ = π(1.0), l₃ = π(1.5)... Total Length L = π(0.5) + π(1.0) + ... (13 terms) L = π[0.5 + 1.0 + ... ] Inside bracket is AP: a = 0.5, d = 0.5, n = 13 Sum = 132[2(0.5) + 12(0.5)] Sum = 132[1 + 6] = 132(7) = 912 Total L = π × 912 L = 227 × 912 L = 11 × 13 = 143 Answer: 143 cm
Q19. 200 logs stacked. 20 bottom, 19 next... How many rows? Top row logs?
AP: 20, 19, 18... Sum = 200 a = 20, d = -1, Sₙ = 200 200 = n2[2(20) + (n - 1)(-1)] 400 = n[40 - n + 1] 400 = n[41 - n] n² - 41n + 400 = 0 Factors of 400 summing to 41: 16 and 25 (n - 16)(n - 25) = 0 n = 16 or n = 25 If n = 25: a₂₅ = 20 + 24(-1) = -4 (Negative logs impossible) So n = 16 Top row (a₁₆) = 20 + 15(-1) = 5 Answer: 16 rows, 5 logs in top row
