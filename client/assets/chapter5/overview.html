Chapter 5: Arithmetic Progressions (Overview)

Introduction

In daily life, we see many patterns. Example: Savings every month, or steps of a ladder. When numbers increase or decrease by a fixed amount, we call it an AP. This chapter helps us calculate large values simply without counting one by one.

Definitions

  • Arithmetic Progression (AP): A list of numbers where difference between two consecutive terms is always same.
  • First Term (a): The number starting the list. It can be any number.
  • Common Difference (d): The fixed number added to get the next term.
  • Note: Common difference 'd' can be positive (increasing), negative (decreasing), or zero.
  • Finite AP: An AP with limited terms (End is known).
  • Infinite AP: An AP that never ends (Last term not known).

Key Points

  • General form of an AP is: a, a + d, a + 2d, a + 3d ...
  • nth Term Formula (General Term):
    aₙ = a + (n - 1)d
  • Sum of First n Terms Formula (1):
    Sₙ = n2 [ 2a + (n - 1)d ]
  • Sum Formula (2) if last term (l) is known:
    Sₙ = n2 ( a + l )
  • Sum of first n positive integers:
    Sₙ = n(n + 1)2

Crux

This whole chapter is about only TWO main formulas. First, always find values of 'a' and 'd' from the question. If they ask "Find 10th term", use the aₙ formula. If they ask "Find Total" or "Sum", use the Sₙ formula. Be careful with 'd' signs. If AP is decreasing (10, 8, 6...), 'd' is negative (-2). If 3 variables are known out of a, d, n, aₙ, you can find the 4th one easily. That is all you need for the exam!
