Chapter 7: Coordinate Geometry Overview
1. Introduction
Coordinate System: A system that uses coordinates to uniquely determine the position of points.
Cartesian Plane: The plane formed by the intersection of two perpendicular number lines (x-axis and y-axis).
Axes: The horizontal line is the x-axis, vertical is the y-axis.
Origin: Intersection of axes, O(0, 0).
Quadrants: Axes divide the plane into four quadrants (I, II, III, IV).
Abscissa (x): Perpendicular distance from y-axis.
Ordinate (y): Perpendicular distance from x-axis.
2. Distance Formula
The distance between any two points P(x₁, y₁) and Q(x₂, y₂) is given by:
d = √[(x₂ - x₁)² + (y₂ - y₁)²]
Note: The distance of a point P(x, y) from the origin O(0, 0) is √(x² + y²).
3. Section Formula
The coordinates of the point P(x, y) which divides the line segment joining the points A(x₁, y₁) and B(x₂, y₂) internally in the ratio m₁ : m₂ are:
P(x, y) = ( m₁x₂ + m₂x₁m₁ + m₂ , m₁y₂ + m₂y₁m₁ + m₂ )
4. Mid-point Formula
If P is the mid-point of AB, then the ratio is 1:1. The coordinates are:
P(x, y) = ( x₁ + x₂2 , y₁ + y₂2 )

6. Key Concepts
1. Collinear Points: Three points A, B, and C are collinear if distance AB + BC = AC.
2. Equidistant: A point P is equidistant from A and B if PA = PB.
3. Quadrilateral Types:
- Parallelogram: Diagonals bisect each other.
- Rectangle: Diagonals bisect each other and apply Pythagoras.
- Rhombus: All sides equal, diagonals bisect at 90°.
