1. Solve the following pair of linear equations by the substitution method.
(i) x + y = 14 ; x – y = 4
Solution:
x + y = 14 ...(1)
x - y = 4 ...(2)
From (2):
x = 4 + y ...(3)
Substitute x in (1):
(4 + y) + y = 14
2y + 4 = 14
2y = 10
y = 5
Substitute y = 5 in (3):
x = 4 + 5 = 9
x = 9, y = 5
(ii) s – t = 3 ; s3 + t2 = 6
Solution:
s - t = 3 ...(1)
s3 + t2 = 6 ...(2)
From (1):
s = 3 + t ...(3)
Substitute s in (2):
3 + t3 + t2 = 6
Multiply by 6:
2(3 + t) + 3t = 36
6 + 2t + 3t = 36
5t = 30
t = 6
Substitute t = 6 in (3):
s = 3 + 6 = 9
s = 9, t = 6
(iv) 0.2x + 0.3y = 1.3 ; 0.4x + 0.5y = 2.3
Solution:
Multiply equations by 10:
2x + 3y = 13 ...(1)
4x + 5y = 23 ...(2)
From (1):
x = 13 - 3y2 ...(3)
Substitute x in (2):
4(13 - 3y2) + 5y = 23
2(13 - 3y) + 5y = 23
26 - 6y + 5y = 23
26 - y = 23
y = 3
Substitute y = 3 in (3):
x = 13 - 92
x = 42 = 2
x = 2, y = 3
(v) √2x + √3y = 0 ; √3x – √8y = 0
Solution:
√2x + √3y = 0 ...(1)
√3x - √8y = 0 ...(2)
From (1): x = -√3y√2 ...(3)
Substitute x in (2):
√3(-√3y√2) - √8y = 0
-3y√2 - 2√2y = 0
Taking y common: y(-3√2 - 2√2) = 0
Since constant term is not zero, y must be 0.
Substitute y = 0 in (3):
x = 0
x = 0, y = 0
2. Solve 2x + 3y = 11 and 2x – 4y = – 24 and hence find the value of ‘m’ for which y = mx + 3.
Solution:
2x + 3y = 11 ...(1)
2x - 4y = -24 ...(2)
Subtract (2) from (1):
(2x + 3y) - (2x - 4y) = 11 - (-24)
7y = 35
y = 5
Substitute y = 5 in (1):
2x + 15 = 11
2x = -4
x = -2
Now find m using y = mx + 3:
5 = m(-2) + 3
5 - 3 = -2m
2 = -2m
m = -1
m = -1
