Chapter 5: Arithmetic Progressions

Textbook Examples Solved

Example 1: For the AP: 32, 12, -12, -32... write first term a and common difference d.
First term a = 32
d = Second term - First term
d = 12 - 32
d = 1 - 32 = -22 = -1
Answer: a = 32, d = -1
Example 2: Which of the following lists of numbers form an AP? If yes, write next two terms.
(i) 4, 10, 16, 22...
10 - 4 = 6
16 - 10 = 6
22 - 16 = 6
Difference is constant (6). Yes, it is an AP.
Next terms: 22 + 6 = 28, 28 + 6 = 34
Answer: Yes; 28, 34
(ii) 1, -1, -3, -5...
-1 - 1 = -2
-3 - (-1) = -2
-5 - (-3) = -2
Difference is constant (-2). Yes, it is an AP.
Next terms: -5 - 2 = -7, -7 - 2 = -9
Answer: Yes; -7, -9
(iii) -2, 2, -2, 2, -2...
2 - (-2) = 4
-2 - 2 = -4
Difference is not constant. No, it is not an AP.
Answer: No
(iv) 1, 1, 1, 2, 2, 2, 3, 3, 3...
1 - 1 = 0
2 - 1 = 1
Difference is not constant. No, it is not an AP.
Answer: No
Example 3: Find the 10th term of the AP: 2, 7, 12...
a = 2, d = 7 - 2 = 5, n = 10
Formula: aₙ = a + (n - 1)d
a₁₀ = 2 + (10 - 1)5
a₁₀ = 2 + 9(5)
a₁₀ = 2 + 45 = 47
Answer: 47
Example 4: Which term of the AP: 21, 18, 15... is -81? Also, is any term 0?
a = 21, d = 18 - 21 = -3
Let nth term be -81.
-81 = 21 + (n - 1)(-3)
-81 - 21 = (n - 1)(-3)
-102 = (n - 1)(-3)
n - 1 = -102 / -3 = 34
n = 35. So, 35th term is -81.

Now check for 0:
0 = 21 + (m - 1)(-3)
-21 = (m - 1)(-3)
m - 1 = 7
m = 8. So, 8th term is 0.
Answer: 35th term is -81; Yes, 8th term is 0
Example 5: Determine the AP whose 3rd term is 5 and 7th term is 9.
a₃ = a + 2d = 5 ...(1)
a₇ = a + 6d = 9 ...(2)
Subtract (1) from (2):
4d = 4 → d = 1
Put d=1 in (1): a + 2(1) = 5 → a = 3
AP is: 3, 4, 5, 6...
Answer: 3, 4, 5, 6...
Example 6: Check whether 301 is a term of 5, 11, 17, 23...
a = 5, d = 11 - 5 = 6
Let 301 be nth term.
301 = 5 + (n - 1)6
296 = (n - 1)6
n - 1 = \(\frac{296}{6}\) = 49.33...
Since n is not a whole number (integer), 301 is not a term.
Answer: No, it is not a term.
Example 7: How many two-digit numbers are divisible by 3?
Two digit numbers: 10 to 99.
First divisible by 3: 12. Last: 99.
List: 12, 15, 18, ..., 99
This is an AP with a = 12, d = 3, aₙ = 99.
99 = 12 + (n - 1)3
87 = (n - 1)3
n - 1 = 29
n = 30
Answer: 30 numbers
Example 8: Find the 11th term from the last term of the AP: 10, 7, 4, ..., -62.
Method: Reverse the AP.
New AP: -62, ..., 4, 7, 10
a = -62, d = 3 (since original d was -3)
Find 11th term: a₁₁ = a + 10d
a₁₁ = -62 + 10(3)
a₁₁ = -62 + 30 = -32
Answer: -32
Example 9: ₹ 1000 invested at 8% SI. Calculate interest at end of each year. Do they form AP? Find interest for 30 years.
Simple Interest = (P × R × T) / 100
Year 1: (1000 × 8 × 1)/100 = ₹ 80
Year 2: (1000 × 8 × 2)/100 = ₹ 160
Year 3: (1000 × 8 × 3)/100 = ₹ 240
List: 80, 160, 240...
Difference is 80 (constant). Yes, it is an AP.
Find interest for 30 years (30th term).
a₃₀ = a + 29d
a₃₀ = 80 + 29(80)
Or simply, I = (1000 × 8 × 30)/100 = 2400.
Using AP: 80 + 2320 = 2400.
Answer: Yes AP; Interest at 30 years = ₹ 2400
Example 10: Rose plants: 23, 21, 19... last row 5. How many rows?
a = 23, d = -2, aₙ = 5
5 = 23 + (n - 1)(-2)
-18 = (n - 1)(-2)
n - 1 = 9
n = 10
Answer: 10 rows
Example 11: Find sum of first 22 terms of AP: 8, 3, -2...
a = 8, d = 3 - 8 = -5, n = 22
Sₙ = n2[2a + (n - 1)d]
S₂₂ = 222[2(8) + 21(-5)]
S₂₂ = 11[16 - 105]
S₂₂ = 11(-89)
S₂₂ = -979
Answer: -979
Example 12: Sum of first 14 terms is 1050, first term is 10. Find 20th term.
S₁₄ = 1050, a = 10, n = 14
1050 = 142[2(10) + 13d]
1050 = 7[20 + 13d]
150 = 20 + 13d
130 = 13d → d = 10
Find a₂₀ = a + 19d
a₂₀ = 10 + 19(10)
a₂₀ = 10 + 190 = 200
Answer: 200
Example 13: How many terms of AP: 24, 21, 18... must be taken so sum is 78?
a = 24, d = -3, Sₙ = 78
78 = n2[2(24) + (n - 1)(-3)]
156 = n[48 - 3n + 3]
156 = n[51 - 3n]
156 = 51n - 3n²
3n² - 51n + 156 = 0
Divide by 3: n² - 17n + 52 = 0
(n - 4)(n - 13) = 0
n = 4 or n = 13
Both values are valid.
Answer: 4 or 13 terms
