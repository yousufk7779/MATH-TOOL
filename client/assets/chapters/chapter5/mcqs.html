Multiple Choice Questions

Arithmetic Progressions (Class 10)

1. The fixed number added to each term in an AP is called:
(A) First term
(B) Common difference
(C) Sum
(D) Last term
2. If the first term is a and common difference is d, the second term is:
(A) a + d
(B) a - d
(C) ad
(D) a / d
3. Which of the following is an AP?
(A) 2, 4, 8, 16
(B) 1, 3, 5, 7
(C) 1, 2, 4, 7
(D) 1, 5, 10, 15
4. The common difference of the AP: 10, 7, 4... is:
(A) 3
(B) -3
(C) 7
(D) 4
5. The nth term formula for an AP is:
(A) a + (n - 1)d
(B) a + nd
(C) 2a + (n - 1)d
(D) a - (n - 1)d
6. In an AP, if a = 5 and d = 2, what is the 10th term?
(A) 20
(B) 23
(C) 25
(D) 27
7. The sum of the first n terms of an AP starting with a and ending with l is:
(A) \(\frac{n}{2}\)(a + l)
(B) n(a + l)
(C) (a + l) / 2
(D) \(\frac{n}{2}\)(a - l)
8. If a, b, c are in AP, then 2b is equal to:
(A) a + c
(B) a - c
(C) ac
(D) a / c
9. The common difference of an AP can be:
(A) Positive
(B) Negative
(C) Zero
(D) All of these
10. The first term of the AP: -5, -1, 3, 7 is:
(A) -5
(B) -1
(C) 1
(D) 7
