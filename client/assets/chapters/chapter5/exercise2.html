Q1. Fill in the blanks. Formula used: aₙ = a + (n - 1)d
(i) a = 7, d = 3, n = 8. Find aₙ.
aₙ = 7 + (8 - 1)3
aₙ = 7 + (7)3
aₙ = 7 + 21
aₙ = 28
Answer: 28
(ii) a = -18, n = 10, aₙ = 0. Find d.
0 = -18 + (10 - 1)d
18 = 9d
d = 18 ÷ 9
d = 2
Answer: d = 2
(iii) d = -3, n = 18, aₙ = -5. Find a.
-5 = a + (18 - 1)(-3)
-5 = a + (17)(-3)
-5 = a - 51
a = -5 + 51
a = 46
Answer: a = 46
(iv) a = -18.9, d = 2.5, aₙ = 3.6. Find n.
3.6 = -18.9 + (n - 1)2.5
3.6 + 18.9 = (n - 1)2.5
22.5 = (n - 1)2.5
n - 1 = 22.5 ÷ 2.5
n - 1 = 9
n = 9 + 1 = 10
Answer: n = 10
(v) a = 3.5, d = 0, n = 105. Find aₙ.
aₙ = 3.5 + (105 - 1)0
aₙ = 3.5 + 0
aₙ = 3.5
Answer: 3.5

Q7. Find 31st term if 11th term is 38 and 16th term is 73.
a₁₁ = a + 10d = 38
a₁₆ = a + 15d = 73
Subtracting: 5d = 35 → d = 7
a + 10(7) = 38
a = 38 - 70 = -32
a₃₁ = a + 30d
a₃₁ = -32 + 30(7)
a₃₁ = -32 + 210 = 178
Answer: 178
Q8. AP of 50 terms. 3rd term = 12, Last term = 106. Find 29th term.
Total terms n = 50. So a₅₀ = 106.
a₃ = a + 2d = 12
a₅₀ = a + 49d = 106
Subtracting: 47d = 94 → d = 2
a + 2(2) = 12 → a = 8
a₂₉ = a + 28d
a₂₉ = 8 + 28(2)
a₂₉ = 8 + 56 = 64
Answer: 64
Q9. 3rd term is 4, 9th term is -8. Which term is 0?
a₃ = a + 2d = 4
a₉ = a + 8d = -8
Subtracting (2)-(1): 6d = -12
d = -2
a + 2(-2) = 4 → a - 4 = 4 → a = 8
Let nth term be 0.
0 = 8 + (n - 1)(-2)
-8 = (n - 1)(-2)
4 = n - 1
n = 5
Answer: 5th term
Q10. 17th term exceeds 10th term by 7. Find common difference.
a₁₇ - a₁₀ = 7
(a + 16d) - (a + 9d) = 7
a + 16d - a - 9d = 7
7d = 7
d = 1
Answer: 1
Q11. Which term of 3, 15, 27, 39... will be 132 more than 54th term?
a = 3, d = 12
a₅₄ = 3 + 53(12)
a₅₄ = 3 + 636 = 639
Required term = 639 + 132 = 771
771 = 3 + (n - 1)12
768 = (n - 1)12
64 = n - 1
n = 65
Answer: 65th term
Q12. Two APs have same common difference. Diff between 100th terms is 100. Diff between 1000th terms?
Let APs be a, a+d... and A, A+d...
Diff between nth terms = (A + (n-1)d) - (a + (n-1)d)
= A - a (constant difference)
Given diff between 100th terms = 100.
So A - a = 100.
Diff between 1000th terms will also be A - a.
So difference is 100.
Answer: 100

Q16. Determine AP: 3rd term is 16, 7th term exceeds 5th by 12.
a₃ = 16
a₇ - a₅ = 12
(a + 6d) - (a + 4d) = 12
2d = 12 → d = 6
a₃ = a + 2d = 16
a + 2(6) = 16
a + 12 = 16 → a = 4
AP: 4, 10, 16, 22...
Answer: 4, 10, 16, 22...
Q17. Find the 20th term from the last term of the AP: 3, 8, 13, ..., 253.
For last term, we reverse the AP.
New AP: 253, ..., 13, 8, 3
a = 253, d = 3 - 8 = -5
Find 20th term (a₂₀).
a₂₀ = a + 19d
a₂₀ = 253 + 19(-5)
a₂₀ = 253 - 95
a₂₀ = 158
Answer: 158
Q18. Sum of 4th and 8th terms is 24. Sum of 6th and 10th terms is 44. Find AP.
a₄ + a₈ = 24
(a + 3d) + (a + 7d) = 24
2a + 10d = 24 → a + 5d = 12 ...(1)
a₆ + a₁₀ = 44
(a + 5d) + (a + 9d) = 44
2a + 14d = 44 → a + 7d = 22 ...(2)
Subtracting (1) from (2):
2d = 10 → d = 5
From (1): a + 5(5) = 12
a + 25 = 12 → a = -13
Terms: -13, -8, -3...
Answer: -13, -8, -3...

Q20. Ramkali saved ₹ 5 first week, increased by ₹ 1.75. nth week ₹ 20.75. Find n.
a = 5, d = 1.75, aₙ = 20.75
20.75 = 5 + (n - 1)1.75
15.75 = (n - 1)1.75
n - 1 = 15.75 ÷ 1.75
n - 1 = 1575 ÷ 175
n - 1 = 9
n = 10
Answer: n = 10
