1. Form the pair of linear equations in the following problems, and find their solutions graphically.
(i) 10 students of Class X took part in a Mathematics quiz. If the number of girls is 4 more than the number of boys, find the number of boys and girls who took part in the quiz.
Given:
Total students = 10
Number of girls is 4 more than the number of boys.
To Find:
Number of boys and girls.
Solution:
Let number of girls = x
Let number of boys = y
According to the first condition:
x + y = 10 ...(1)
According to the second condition:
x = y + 4
x - y = 4 ...(2)
Observation for Graph:
For equation (1): x + y = 10
x 4 6
y 6 4
For equation (2): x - y = 4
x 4 6
y 0 2
Plotting these points, the lines intersect at (7, 3).
Number of Girls = 7, Number of Boys = 3
(ii) 5 pencils and 7 pens together cost 50, whereas 7 pencils and 5 pens together cost 46. Find the cost of one pencil and that of one pen.
Given:
Cost of 5 pencils and 7 pens = 50
Cost of 7 pencils and 5 pens = 46
To Find:
Cost of one pencil and one pen.
Solution:
Let the cost of 1 pencil = x
Let the cost of 1 pen = y
According to the conditions:
5x + 7y = 50 ...(1)
7x + 5y = 46 ...(2)
Observation for Graph:
For equation (1): 5x + 7y = 50
x 10 3
y 0 5
For equation (2): 7x + 5y = 46
x 8 3
y -2 5
The lines intersect at (3, 5).
Cost of Pencil = 3, Cost of Pen = 5
2. On comparing the ratios \(\frac{a}{a}\), \(\frac{b}{b}\) and \(\frac{c}{c}\), find out whether the lines representing the following pairs of linear equations intersect at a point, are parallel or coincident:
(i) 5x 4y + 8 = 0; 7x + 6y 9 = 0
Solution:
a = 5, b = -4
a = 7, b = 6
\(\frac{a}{a}\) = \(\frac{5}{7}\)
\(\frac{b}{b}\) = \(\frac{-4}{6}\) = \(\frac{-2}{3}\)
Since \(\frac{a}{a}\) \(\frac{b}{b}\)
Lines intersect at a single point.
(ii) 9x + 3y + 12 = 0; 18x + 6y + 24 = 0
Solution:
\(\frac{a}{a}\) = \(\frac{9}{18}\) = \(\frac{1}{2}\)
\(\frac{b}{b}\) = \(\frac{3}{6}\) = \(\frac{1}{2}\)
\(\frac{c}{c}\) = \(\frac{12}{24}\) = \(\frac{1}{2}\)
Since all ratios are equal.
Lines are coincident (Infinite solutions).
(iii) 6x 3y + 10 = 0; 2x y + 9 = 0
Solution:
\(\frac{a}{a}\) = \(\frac{6}{2}\) = 3
\(\frac{b}{b}\) = \(\frac{-3}{-1}\) = 3
\(\frac{c}{c}\) = \(\frac{10}{9}\)
Since \(\frac{a}{a}\) = \(\frac{b}{b}\) \(\frac{c}{c}\)
Lines are parallel (No solution).
3. On comparing the ratios \(\frac{a}{a}\), \(\frac{b}{b}\) and \(\frac{c}{c}\), find out whether the following pair of linear equations are consistent or inconsistent.
(i) 3x + 2y = 5 ; 2x 3y = 7
Solution:
\(\frac{a}{a}\) = \(\frac{3}{2}\)
\(\frac{b}{b}\) = \(\frac{2}{-3}\)
Since \(\frac{a}{a}\) \(\frac{b}{b}\)
Consistent (Unique Solution).
(ii) 2x 3y = 8 ; 4x 6y = 9
Solution:
\(\frac{a}{a}\) = \(\frac{2}{4}\) = \(\frac{1}{2}\)
\(\frac{b}{b}\) = \(\frac{-3}{-6}\) = \(\frac{1}{2}\)
\(\frac{c}{c}\) = \(\frac{8}{9}\)
Since \(\frac{a}{a}\) = \(\frac{b}{b}\) \(\frac{c}{c}\)
Inconsistent (Parallel lines).
(iii) \(\frac{3}{2}\)x + \(\frac{5}{3}\)y = 7 ; 9x 10y = 14
Solution:
\(\frac{a}{a}\) = \(\frac{3/2}{9}\) = \(\frac{1}{6}\)
\(\frac{b}{b}\) = \(\frac{5/3}{-10}\) = \(\frac{-1}{6}\)
Since \(\frac{a}{a}\) \(\frac{b}{b}\)
Consistent.
(iv) 5x 3y = 11 ; 10x + 6y = 22
Solution:
\(\frac{a}{a}\) = \(\frac{5}{-10}\) = \(\frac{-1}{2}\)
\(\frac{b}{b}\) = \(\frac{-3}{6}\) = \(\frac{-1}{2}\)
\(\frac{c}{c}\) = \(\frac{11}{-22}\) = \(\frac{-1}{2}\)
Since \(\frac{a}{a}\) = \(\frac{b}{b}\) = \(\frac{c}{c}\)
Consistent (\(\frac{Dependent}{Coincident}\)).
(v) \(\frac{4}{3}\)x + 2y = 8 ; 2x + 3y = 12
Solution:
\(\frac{a}{a}\) = \(\frac{4/3}{2}\) = \(\frac{2}{3}\)
\(\frac{b}{b}\) = \(\frac{2}{3}\)
\(\frac{c}{c}\) = \(\frac{8}{12}\) = \(\frac{2}{3}\)
Since \(\frac{a}{a}\) = \(\frac{b}{b}\) = \(\frac{c}{c}\)
Consistent (\(\frac{Dependent}{Coincident}\)).
4. Which of the following pairs of linear equations are \(\frac{consistent}{inconsistent}\)? If consistent, obtain the solution graphically:
(i) x + y = 5, 2x + 2y = 10
Solution:
\(\frac{a}{a}\) = \(\frac{1}{2}\), \(\frac{b}{b}\) = \(\frac{1}{2}\), \(\frac{c}{c}\) = \(\frac{5}{10}\) = \(\frac{1}{2}\)
Since ratios are equal, the lines are coincident.
Consistent (Infinitely many solutions).
(ii) x y = 8, 3x 3y = 16
Solution:
\(\frac{a}{a}\) = \(\frac{1}{3}\)
\(\frac{b}{b}\) = \(\frac{-1}{-3}\) = \(\frac{1}{3}\)
\(\frac{c}{c}\) = \(\frac{8}{16}\) = \(\frac{1}{2}\)
Since \(\frac{a}{a}\) = \(\frac{b}{b}\) \(\frac{c}{c}\)
Inconsistent (Parallel lines).
(iii) 2x + y 6 = 0, 4x 2y 4 = 0
Solution:
\(\frac{a}{a}\) = \(\frac{2}{4}\) = \(\frac{1}{2}\)
\(\frac{b}{b}\) = \(\frac{1}{-2}\)
Since \(\frac{a}{a}\) \(\frac{b}{b}\)
Lines intersect at a unique point.
Consistent (Solution: x=2, y=2).
(iv) 2x 2y 2 = 0, 4x 4y 5 = 0
Solution:
\(\frac{a}{a}\) = \(\frac{2}{4}\) = \(\frac{1}{2}\)
\(\frac{b}{b}\) = \(\frac{-2}{-4}\) = \(\frac{1}{2}\)
\(\frac{c}{c}\) = \(\frac{-2}{-5}\) = \(\frac{2}{5}\)
Since \(\frac{a}{a}\) = \(\frac{b}{b}\) \(\frac{c}{c}\)
Inconsistent (Parallel lines).
5. Half the perimeter of a rectangular garden, whose length is 4 m more than its width, is 36 m. Find the dimensions of the garden.
Solution:
Let length = x meters, width = y meters
Length is 4 more than width: x - y = 4 ...(1)
Half perimeter = 36: x + y = 36 ...(2)
Graphical Solution:
From (1): x = y + 4
x812
y48
From (2): y = 36 - x
x1620
y2016
Lines intersect at (20, 16).
Length = 20 m, Width = 16 m
6. Given the linear equation 2x + 3y 8 = 0, write another linear equation in two variables such that the geometrical representation of the pair so formed is:
(i) intersecting lines
Solution:
Condition: \(\frac{a}{a}\) \(\frac{b}{b}\)
Example: 3x + 2y - 7 = 0
(ii) parallel lines
Solution:
Condition: \(\frac{a}{a}\) = \(\frac{b}{b}\) \(\frac{c}{c}\)
Example: 2x + 3y - 12 = 0
(iii) coincident lines
Solution:
Condition: \(\frac{a}{a}\) = \(\frac{b}{b}\) = \(\frac{c}{c}\)
Example: 4x + 6y - 16 = 0
