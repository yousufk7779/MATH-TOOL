Chapter 3: Linear Equations
1. Introduction

A pair of linear equations in two variables x and y represents two straight lines in a 2D plane. We can solve them to find if they intersect at a point (unique solution), are parallel (no solution), or coincident (infinitely many solutions).

2. Standard Form
The general form for a pair of linear equations is:
1. a₁x + b₁y + c₁ = 0
2. a₂x + b₂y + c₂ = 0
where a₁, b₁, c₁, a₂, b₂, c₂ are real numbers.
3. Conditions for Consistency
Case 1: Intersecting Lines
Condition: a₁a₂ ≠ b₁b₂
Result: Exactly one solution (Unique). The system is Consistent.
Case 2: Coincident Lines
Condition: a₁a₂ = b₁b₂ = c₁c₂
Result: Infinitely many solutions. The system is Consistent (Dependent).
Case 3: Parallel Lines
Condition: a₁a₂ = b₁b₂ ≠ c₁c₂
Result: No solution. The system is Inconsistent.
4. Methods of Solution
Graphical Method: Plot both lines on a graph paper and find the point of intersection.
Substitution Method: Compute the value of one variable from one equation and substitute it into the other equation.
Elimination Method: Multiply equations by constants to make the coefficients of one variable equal, then add or subtract to eliminate that variable.
