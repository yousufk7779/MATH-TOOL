1. The general form of a linear equation in two variables x and y is:
(a) ax + by + c = 0
(b) ax² + bx + c = 0
(c) ax + b = 0
(d) None of these
Answer: (a) ax + by + c = 0
2. If the lines are parallel, then the pair of equations has how many solutions?
(a) Unique solution
(b) No solution
(c) Infinitely many solutions
(d) Two solutions
Answer: (b) No solution
3. If the lines intersect at a single point, then the pair of equations has:
(a) A unique solution
(b) No solution
(c) Infinitely many solutions
(d) No conclusion
Answer: (a) A unique solution
4. If the lines are coincident (lie on top of each other), then the pair of equations has:
(a) A unique solution
(b) No solution
(c) Infinitely many solutions
(d) One solution
Answer: (c) Infinitely many solutions
5. A pair of linear equations which has no solution is called:
(a) Consistent
(b) Inconsistent
(c) Dependent
(d) None of these
Answer: (b) Inconsistent
6. A pair of linear equations which has at least one solution is called:
(a) Consistent
(b) Inconsistent
(c) Parallel
(d) None of these
Answer: (a) Consistent
7. The graph of a linear equation ax + by + c = 0 is always a:
(a) Circle
(b) Straight line
(c) Parabola
(d) Point
Answer: (b) Straight line
8. If a₁/a₂ ≠ b₁/b₂, then the system of linear equations has:
(a) A unique solution
(b) No solution
(c) Infinite solutions
(d) Two solutions
Answer: (a) A unique solution
9. If a₁/a₂ = b₁/b₂ ≠ c₁/c₂, then the system of linear equations has:
(a) A unique solution
(b) No solution
(c) Infinite solutions
(d) Many solutions
Answer: (b) No solution
10. If a₁/a₂ = b₁/b₂ = c₁/c₂, then the system of linear equations has:
(a) A unique solution
(b) No solution
(c) Infinitely many solutions
(d) Two solutions
Answer: (c) Infinitely many solutions
