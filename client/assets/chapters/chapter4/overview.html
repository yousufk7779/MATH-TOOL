Basic Introductions
» A quadratic equation is a second-degree polynomial equation in a single variable x, typically written in the standard form ax² + bx + c = 0.
Definitions
» Roots: The values of x that satisfy the equation.
» Discriminant (D): Evaluated as b² - 4ac, which determines the nature of the roots.
Key points
» Solved using methods like factorization, completing the square, or the quadratic formula x = (-b ± √D) / 2a.
Crux
» If D > 0, roots are real and distinct.
» If D = 0, roots are real and equal.
» If D < 0, there are no real roots.
Summary
» Quadratic equations naturally model situations involving gravity, area, and parabolic trajectories.
» The discriminant is the fastest tool to check root validity.
