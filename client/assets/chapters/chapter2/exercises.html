Exercise 2.2
1. Find the zeroes of the following quadratic polynomials and verify the relationship between the zeroes and the coefficients.
(i) x - 2x - 8
Given:
Polynomial p(x) = x - 2x - 8
To Find:
Zeroes of p(x) and verify the relationship with coefficients.
Solution:
To find zeroes, we set p(x) = 0.
We split the middle term (-2x) such that the product is -8x and sum is -2x.
The terms are -4x and +2x.
x - 4x + 2x - 8 = 0
Taking x common from first two terms and 2 from last two terms:
x(x - 4) + 2(x - 4) = 0
(x - 4)(x + 2) = 0
Either x - 4 = 0 or x + 2 = 0
So, x = 4 or x = -2
Zeroes are = 4 and = -2.
Verification:
Comparing given polynomial with ax + bx + c:
a = 1, b = -2, c = -8
1. Sum of zeroes:
+ = 4 + (-2) = 2
Formula: -\(\frac{b}{a}\) = -(-2)/1 = 2
Hence, + = -\(\frac{b}{a}\)
2. Product of zeroes:
= 4 (-2) = -8
Formula: \(\frac{c}{a}\) = -\(\frac{8}{1}\) = -8
Hence, = \(\frac{c}{a}\)
Relationship verified.
(ii) 4s - 4s + 1
Solution:
Let p(s) = 4s - 4s + 1
To find zeroes, p(s) = 0
Splitting the middle term (-4s) into -2s and -2s:
4s - 2s - 2s + 1 = 0
2s(2s - 1) - 1(2s - 1) = 0
(2s - 1)(2s - 1) = 0
2s - 1 = 0 => s = \(\frac{1}{2}\)
Zeroes are = \(\frac{1}{2}\) and = \(\frac{1}{2}\).
Verification:
Here a = 4, b = -4, c = 1
Sum ( + ) = \(\frac{1}{2}\) + \(\frac{1}{2}\) = 1
Formula: -\(\frac{b}{a}\) = -(-4)/4 = 1
Product () = \(\frac{1}{2}\) \(\frac{1}{2}\) = \(\frac{1}{4}\)
Formula: \(\frac{c}{a}\) = \(\frac{1}{4}\)
Relationship verified.
(iii) 6x - 3 - 7x
Solution:
Rearranging in standard form: 6x - 7x - 3
To find zeroes, 6x - 7x - 3 = 0
Splitting middle term (-7x) into -9x and +2x:
6x - 9x + 2x - 3 = 0
3x(2x - 3) + 1(2x - 3) = 0
(2x - 3)(3x + 1) = 0
2x - 3 = 0 => x = \(\frac{3}{2}\)
3x + 1 = 0 => x = \(\frac{-1}{3}\)
Zeroes are = \(\frac{3}{2}\) and = \(\frac{-1}{3}\).
Verification:
Here a = 6, b = -7, c = -3
Sum = \(\frac{3}{2}\) + (\(\frac{-1}{3}\)) = \(\frac{9 - 2}{6}\) = \(\frac{7}{6}\)
Formula: -\(\frac{b}{a}\) = -(-7)/6 = \(\frac{7}{6}\)
Product = \(\frac{3}{2}\) (\(\frac{-1}{3}\)) = \(\frac{-3}{6}\) = \(\frac{-1}{2}\)
Formula: \(\frac{c}{a}\) = -\(\frac{3}{6}\) = \(\frac{-1}{2}\)
Relationship verified.
(iv) 4u + 8u
Solution:
To find zeroes, 4u + 8u = 0
Taking 4u common:
4u(u + 2) = 0
4u = 0 => u = 0
u + 2 = 0 => u = -2
Zeroes are 0 and -2.
Verification:
Here a = 4, b = 8, c = 0
Sum = 0 + (-2) = -2
Formula: -\(\frac{b}{a}\) = -\(\frac{8}{4}\) = -2
Product = 0 (-2) = 0
Formula: \(\frac{c}{a}\) = \(\frac{0}{4}\) = 0
Relationship verified.
(v) t - 15
Solution:
To find zeroes, t - 15 = 0
Using identity a - b = (a-b)(a+b):
t - (15) = 0
(t - 15)(t + 15) = 0
t = 15 OR t = -15
Zeroes are 15 and -15.
Verification:
Here a = 1, b = 0, c = -15
Sum = 15 + (-15) = 0
Formula: -\(\frac{b}{a}\) = -\(\frac{0}{1}\) = 0
Product = 15 (-15) = -15
Formula: \(\frac{c}{a}\) = -\(\frac{15}{1}\) = -15
Relationship verified.
(vi) 3x - x - 4
Solution:
To find zeroes, 3x - x - 4 = 0
Splitting middle term (-x) into -4x and +3x:
3x - 4x + 3x - 4 = 0
x(3x - 4) + 1(3x - 4) = 0
(3x - 4)(x + 1) = 0
3x - 4 = 0 => x = \(\frac{4}{3}\)
x + 1 = 0 => x = -1
Zeroes are \(\frac{4}{3}\) and -1.
Verification:
Here a = 3, b = -1, c = -4
Sum = \(\frac{4}{3}\) + (-1) = \(\frac{1}{3}\)
Formula: -\(\frac{b}{a}\) = -(-1)/3 = \(\frac{1}{3}\)
Product = \(\frac{4}{3}\) (-1) = \(\frac{-4}{3}\)
Formula: \(\frac{c}{a}\) = \(\frac{-4}{3}\)
Relationship verified.

= k[x - \(\frac{1}{4}\)x + (-1)]

4x - x - 4

Example 2. Find the zeroes of the quadratic polynomial x + 7x + 10, and verify the relationship between the zeroes and the coefficients.
Solution:
Let p(x) = x + 7x + 10
To find zeroes, put p(x) = 0
By splitting the middle term:
x + 2x + 5x + 10 = 0
x(x + 2) + 5(x + 2) = 0
(x + 2)(x + 5) = 0
x + 2 = 0 OR x + 5 = 0
x = -2 OR x = -5
Zeroes are = -2 and = -5.
Verification:
Comparing with ax + bx + c: a = 1, b = 7, c = 10
Sum of zeroes:
+ = -2 + (-5) = -7
-\(\frac{b}{a}\) = -\(\frac{7}{1}\) = -7
Matches.
Product of zeroes:
= (-2) (-5) = 10
\(\frac{c}{a}\) = \(\frac{10}{1}\) = 10
Matches.
Relationship verified.
Example 3. Find the zeroes of the polynomial x - 3 and verify the relationship between the zeroes and the coefficients.
Solution:
Let p(x) = x - 3
To find zeroes, put p(x) = 0
x - 3 = 0
x - (3) = 0
Using identity a - b = (a - b)(a + b)
(x - 3)(x + 3) = 0
x - 3 = 0 OR x + 3 = 0
x = 3 OR x = -3
Zeroes are = 3 and = -3.
Verification:
Comparing with ax + bx + c: a = 1, b = 0 (no x term), c = -3
Sum of zeroes:
+ = 3 + (-3) = 0
-\(\frac{b}{a}\) = -\(\frac{0}{1}\) = 0
Product of zeroes:
= 3 (-3) = -3
\(\frac{c}{a}\) = -\(\frac{3}{1}\) = -3
Relationship verified.

Example 5. Verify that 3, -1, - are the zeroes of the cubic polynomial p(x) = 3x - 5x - 11x - 3, and then verify the relationship between the zeroes and the coefficients.
Solution:
Given polynomial: p(x) = 3x - 5x - 11x - 3
Checking 3:
p(3) = 3(3) - 5(3) - 11(3) - 3
= 3(27) - 5(9) - 33 - 3
= 81 - 45 - 36
= 81 - 81 = 0
Checking -1:
p(-1) = 3(-1) - 5(-1) - 11(-1) - 3
= -3 - 5 + 11 - 3
= -11 + 11 = 0
Checking -\(\frac{1}{3}\):
p(-\(\frac{1}{3}\)) = 3(-\(\frac{1}{3}\)) - 5(-\(\frac{1}{3}\)) - 11(-\(\frac{1}{3}\)) - 3
= 3(-\(\frac{1}{27}\)) - 5(\(\frac{1}{9}\)) + \(\frac{11}{3}\) - 3
= -\(\frac{1}{9}\) - \(\frac{5}{9}\) + \(\frac{33}{9}\) - \(\frac{27}{9}\)
= (-1 - 5 + 33 - 27) / 9
= \(\frac{0}{9}\) = 0
Thus, 3, -1, and -\(\frac{1}{3}\) are indeed zeroes of p(x).
