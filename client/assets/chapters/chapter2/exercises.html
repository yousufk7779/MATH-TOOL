EXERCISE2
Chapter 2: Polynomials - Exercise 2.2
1. Find the zeroes of the following quadratic polynomials and verify the relationship between the zeroes and the coefficients.
(i) x² - 2x - 8
Given:
Polynomial p(x) = x² - 2x - 8
To Find:
Zeroes of p(x) and verify the relationship with coefficients.
Solution:
To find zeroes, we set p(x) = 0.
We split the middle term (-2x) such that the product is -8x² and sum is -2x.
The terms are -4x and +2x.
x² - 4x + 2x - 8 = 0
Taking x common from first two terms and 2 from last two terms:
x(x - 4) + 2(x - 4) = 0
(x - 4)(x + 2) = 0
Either x - 4 = 0 or x + 2 = 0
So, x = 4 or x = -2
Zeroes are α = 4 and β = -2.
Verification:
Comparing given polynomial with ax² + bx + c:
a = 1, b = -2, c = -8
1. Sum of zeroes:
α + β = 4 + (-2) = 2
Formula: -b/a = -(-2)/1 = 2
Hence, α + β = -b/a
2. Product of zeroes:
α × β = 4 × (-2) = -8
Formula: c/a = -8/1 = -8
Hence, αβ = c/a
Relationship verified.
(ii) 4s² - 4s + 1
Solution:
Let p(s) = 4s² - 4s + 1
To find zeroes, p(s) = 0
Splitting the middle term (-4s) into -2s and -2s:
4s² - 2s - 2s + 1 = 0
2s(2s - 1) - 1(2s - 1) = 0
(2s - 1)(2s - 1) = 0
2s - 1 = 0 => s = 12
Zeroes are α = 12 and β = 12.
Verification:
Here a = 4, b = -4, c = 1
Sum (α + β) = 12 + 12 = 1
Formula: -b/a = -(-4)/4 = 1
Product (αβ) = 12 × 12 = 14
Formula: c/a = 14
Relationship verified.
(iii) 6x² - 3 - 7x
Solution:
Rearranging in standard form: 6x² - 7x - 3
To find zeroes, 6x² - 7x - 3 = 0
Splitting middle term (-7x) into -9x and +2x:
6x² - 9x + 2x - 3 = 0
3x(2x - 3) + 1(2x - 3) = 0
(2x - 3)(3x + 1) = 0
2x - 3 = 0 => x = 32
3x + 1 = 0 => x = -13
Zeroes are α = 32 and β = -13.
Verification:
Here a = 6, b = -7, c = -3
Sum = 32 + (-13) = 9 - 26 = 76
Formula: -b/a = -(-7)/6 = 76
Product = 32 × (-13) = -36 = -12
Formula: c/a = -3/6 = -12
Relationship verified.
(iv) 4u² + 8u
Solution:
To find zeroes, 4u² + 8u = 0
Taking 4u common:
4u(u + 2) = 0
4u = 0 => u = 0
u + 2 = 0 => u = -2
Zeroes are 0 and -2.
Verification:
Here a = 4, b = 8, c = 0
Sum = 0 + (-2) = -2
Formula: -b/a = -8/4 = -2
Product = 0 × (-2) = 0
Formula: c/a = 0/4 = 0
Relationship verified.
(v) t² - 15
Solution:
To find zeroes, t² - 15 = 0
Using identity a² - b² = (a-b)(a+b):
t² - (√15)² = 0
(t - √15)(t + √15) = 0
t = √15 OR t = -√15
Zeroes are √15 and -√15.
Verification:
Here a = 1, b = 0, c = -15
Sum = √15 + (-√15) = 0
Formula: -b/a = -0/1 = 0
Product = √15 × (-√15) = -15
Formula: c/a = -15/1 = -15
Relationship verified.
(vi) 3x² - x - 4
Solution:
To find zeroes, 3x² - x - 4 = 0
Splitting middle term (-x) into -4x and +3x:
3x² - 4x + 3x - 4 = 0
x(3x - 4) + 1(3x - 4) = 0
(3x - 4)(x + 1) = 0
3x - 4 = 0 => x = 43
x + 1 = 0 => x = -1
Zeroes are 43 and -1.
Verification:
Here a = 3, b = -1, c = -4
Sum = 43 + (-1) = 13
Formula: -b/a = -(-1)/3 = 13
Product = 43 × (-1) = -43
Formula: c/a = -43
Relationship verified.

Example 2. Find the zeroes of the quadratic polynomial x² + 7x + 10, and verify the relationship between the zeroes and the coefficients.
Solution:
Let p(x) = x² + 7x + 10
To find zeroes, put p(x) = 0
By splitting the middle term:
x² + 2x + 5x + 10 = 0
x(x + 2) + 5(x + 2) = 0
(x + 2)(x + 5) = 0
x + 2 = 0 OR x + 5 = 0
x = -2 OR x = -5
Zeroes are α = -2 and β = -5.
Verification:
Comparing with ax² + bx + c: a = 1, b = 7, c = 10
Sum of zeroes:
α + β = -2 + (-5) = -7
-b/a = -7/1 = -7
Matches.
Product of zeroes:
α × β = (-2) × (-5) = 10
c/a = 10/1 = 10
Matches.
Relationship verified.
Example 3. Find the zeroes of the polynomial x² - 3 and verify the relationship between the zeroes and the coefficients.
Solution:
Let p(x) = x² - 3
To find zeroes, put p(x) = 0
x² - 3 = 0
x² - (√3)² = 0
Using identity a² - b² = (a - b)(a + b)
(x - √3)(x + √3) = 0
x - √3 = 0 OR x + √3 = 0
x = √3 OR x = -√3
Zeroes are α = √3 and β = -√3.
Verification:
Comparing with ax² + bx + c: a = 1, b = 0 (no x term), c = -3
Sum of zeroes:
α + β = √3 + (-√3) = 0
-b/a = -0/1 = 0
Product of zeroes:
α × β = √3 × (-√3) = -3
c/a = -3/1 = -3
Relationship verified.
Example 4. Find a quadratic polynomial, the sum and product of whose zeroes are -3 and 2, respectively.
Solution:
Given:
Sum of zeroes (S) = -3
Product of zeroes (P) = 2
A quadratic polynomial is given by:
p(x) = k[x² - (Sum of zeroes)x + (Product of zeroes)]
p(x) = k[x² - (-3)x + 2]
p(x) = k[x² + 3x + 2]
For k = 1, the polynomial is:
x² + 3x + 2
Example 5. Verify that 3, -1, -⅓ are the zeroes of the cubic polynomial p(x) = 3x³ - 5x² - 11x - 3, and then verify the relationship between the zeroes and the coefficients.
Solution:
Given polynomial: p(x) = 3x³ - 5x² - 11x - 3
Checking 3:
p(3) = 3(3)³ - 5(3)² - 11(3) - 3
= 3(27) - 5(9) - 33 - 3
= 81 - 45 - 36
= 81 - 81 = 0
Checking -1:
p(-1) = 3(-1)³ - 5(-1)² - 11(-1) - 3
= -3 - 5 + 11 - 3
= -11 + 11 = 0
Checking -1/3:
p(-1/3) = 3(-1/3)³ - 5(-1/3)² - 11(-1/3) - 3
= 3(-1/27) - 5(1/9) + 11/3 - 3
= -1/9 - 5/9 + 33/9 - 27/9
= (-1 - 5 + 33 - 27) / 9
= 0 / 9 = 0
Thus, 3, -1, and -1/3 are indeed zeroes of p(x).
Verification of Relationship:
Comparing with ax³ + bx² + cx + d:
a = 3, b = -5, c = -11, d = -3
Let α = 3, β = -1, γ = -1/3
1. Sum of zeroes (α + β + γ):
3 + (-1) + (-1/3) = 2 - 1/3 = 53
-b/a = -(-5)/3 = 53
2. Sum of product of zeroes taken two at a time (αβ + βγ + γα):
(3)(-1) + (-1)(-1/3) + (-1/3)(3)
= -3 + 1/3 - 1
= -4 + 1/3 = -113
c/a = -113
3. Product of zeroes (αβγ):
(3) × (-1) × (-1/3) = 1
-d/a = -(-3)/3 = 3/3 = 1
Relationship verified.
