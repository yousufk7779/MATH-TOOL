1. Find the zeroes of the following quadratic polynomials and verify the relationship between the zeroes and the coefficients.
(i) x² - 2x - 8
Given:
Polynomial p(x) = x² - 2x - 8
To Find:
Zeroes of p(x) and verify the relationship with coefficients.
Solution:
To find zeroes, we set p(x) = 0.
We split the middle term (-2x) such that the product is -8x² and sum is -2x.
The terms are -4x and +2x.
x² - 4x + 2x - 8 = 0
Taking x common from first two terms and 2 from last two terms:
x(x - 4) + 2(x - 4) = 0
(x - 4)(x + 2) = 0
Either x - 4 = 0 or x + 2 = 0
So, x = 4 or x = -2
Zeroes are α = 4 and β = -2.
Verification:
Comparing given polynomial with ax² + bx + c:
a = 1, b = -2, c = -8
1. Sum of zeroes:
α + β = 4 + (-2) = 2
Formula: -\(\frac{b}{a}\) = -(-2)/1 = 2
Hence, α + β = -\(\frac{b}{a}\)
2. Product of zeroes:
α × β = 4 × (-2) = -8
Formula: \(\frac{c}{a}\) = -\(\frac{8}{1}\) = -8
Hence, αβ = \(\frac{c}{a}\)
Relationship verified.
(ii) 4s² - 4s + 1
Solution:
Let p(s) = 4s² - 4s + 1
To find zeroes, p(s) = 0
Splitting the middle term (-4s) into -2s and -2s:
4s² - 2s - 2s + 1 = 0
2s(2s - 1) - 1(2s - 1) = 0
(2s - 1)(2s - 1) = 0
2s - 1 = 0 => s = \(\frac{1}{2}\)
Zeroes are α = \(\frac{1}{2}\) and β = \(\frac{1}{2}\).
Verification:
Here a = 4, b = -4, c = 1
Sum (α + β) = \(\frac{1}{2}\) + \(\frac{1}{2}\) = 1
Formula: -\(\frac{b}{a}\) = -(-4)/4 = 1
Product (αβ) = \(\frac{1}{2}\) × \(\frac{1}{2}\) = \(\frac{1}{4}\)
Formula: \(\frac{c}{a}\) = \(\frac{1}{4}\)
Relationship verified.
(iii) 6x² - 3 - 7x
Solution:
Rearranging in standard form: 6x² - 7x - 3
To find zeroes, 6x² - 7x - 3 = 0
Splitting middle term (-7x) into -9x and +2x:
6x² - 9x + 2x - 3 = 0
3x(2x - 3) + 1(2x - 3) = 0
(2x - 3)(3x + 1) = 0
2x - 3 = 0 => x = \(\frac{3}{2}\)
3x + 1 = 0 => x = \(\frac{-1}{3}\)
Zeroes are α = \(\frac{3}{2}\) and β = \(\frac{-1}{3}\).
Verification:
Here a = 6, b = -7, c = -3
Sum = \(\frac{3}{2}\) + (\(\frac{-1}{3}\)) = \(\frac{9 - 2}{6}\) = \(\frac{7}{6}\)
Formula: -\(\frac{b}{a}\) = -(-7)/6 = \(\frac{7}{6}\)
Product = \(\frac{3}{2}\) × (\(\frac{-1}{3}\)) = \(\frac{-3}{6}\) = \(\frac{-1}{2}\)
Formula: \(\frac{c}{a}\) = -\(\frac{3}{6}\) = \(\frac{-1}{2}\)
Relationship verified.
(iv) 4u² + 8u
Solution:
To find zeroes, 4u² + 8u = 0
Taking 4u common:
4u(u + 2) = 0
4u = 0 => u = 0
u + 2 = 0 => u = -2
Zeroes are 0 and -2.
Verification:
Here a = 4, b = 8, c = 0
Sum = 0 + (-2) = -2
Formula: -\(\frac{b}{a}\) = -\(\frac{8}{4}\) = -2
Product = 0 × (-2) = 0
Formula: \(\frac{c}{a}\) = \(\frac{0}{4}\) = 0
Relationship verified.
(v) t² - 15
Solution:
To find zeroes, t² - 15 = 0
Using identity a² - b² = (a-b)(a+b):
t² - (√15)² = 0
(t - √15)(t + √15) = 0
t = √15 OR t = -√15
Zeroes are √15 and -√15.
Verification:
Here a = 1, b = 0, c = -15
Sum = √15 + (-√15) = 0
Formula: -\(\frac{b}{a}\) = -\(\frac{0}{1}\) = 0
Product = √15 × (-√15) = -15
Formula: \(\frac{c}{a}\) = -\(\frac{15}{1}\) = -15
Relationship verified.
(vi) 3x² - x - 4
Solution:
To find zeroes, 3x² - x - 4 = 0
Splitting middle term (-x) into -4x and +3x:
3x² - 4x + 3x - 4 = 0
x(3x - 4) + 1(3x - 4) = 0
(3x - 4)(x + 1) = 0
3x - 4 = 0 => x = \(\frac{4}{3}\)
x + 1 = 0 => x = -1
Zeroes are \(\frac{4}{3}\) and -1.
Verification:
Here a = 3, b = -1, c = -4
Sum = \(\frac{4}{3}\) + (-1) = \(\frac{1}{3}\)
Formula: -\(\frac{b}{a}\) = -(-1)/3 = \(\frac{1}{3}\)
Product = \(\frac{4}{3}\) × (-1) = \(\frac{-4}{3}\)
Formula: \(\frac{c}{a}\) = \(\frac{-4}{3}\)
Relationship verified.
2. Find a quadratic polynomial each with the given numbers as the sum and product of its zeroes respectively.
(i) \(\frac{1}{4}\), -1
Solution:
Given: Sum of zeroes (S) = \(\frac{1}{4}\), Product of zeroes (P) = -1
Formula for quadratic polynomial: k[x² - Sx + P]
Substituting values:
= k[x² - \(\frac{1}{4}\)x + (-1)]
To remove the denominator, let k = 4:
= 4(x² - \(\frac{x}{4}\) - 1)
4x² - x - 4

To remove the denominator, let k = 3:
= 3(x² - √2x + \(\frac{1}{3}\))
3x² - 3√2x + 1
