1. The value of \(9 \sec^2 A - 9 \tan^2 A\) is:
(A) 1 (B) 9 (C) 8 (D) 0
Correct Option: (B)
2. The value of \((1 + \tan \theta + \sec \theta)(1 + \cot \theta - \text{cosec } \theta)\) is:
(A) 0 (B) 1 (C) 2 (D) -1
Correct Option: (C)
3. \((\sec A + \tan A)(1 - \sin A)\) is equal to:
(A) \(\sec A\) (B) \(\sin A\) (C) \(\text{cosec } A\) (D) \(\cos A\)
Correct Option: (D)
4. The value of \(\frac{1 + \tan^2 A}{1 + \cot^2 A}\) is:
(A) \(\sec^2 A\) (B) -1 (C) \(\cot^2 A\) (D) \(\tan^2 A\)
Correct Option: (D)
5. If \(\sin A = \frac{3}{4}\), then \(\cos A\) is:
(A) \(\frac{4}{3}\) (B) \(\frac{\sqrt{7}}{4}\) (C) \(\frac{3}{\sqrt{7}}\) (D) \(\frac{\sqrt{7}}{3}\)
Correct Option: (B)
6. If \(4 \tan A = 3\), then the value of \(\frac{4 \sin \theta - \cos \theta}{4 \sin \theta + \cos \theta}\) is:
(A) \(\frac{2}{3}\) (B) \(\frac{1}{3}\) (C) \(\frac{1}{2}\) (D) \(\frac{3}{4}\)
Correct Option: (C)
7. The value of \(\sin 60^\circ \cos 30^\circ + \sin 30^\circ \cos 60^\circ\) is:
(A) 0 (B) 1 (C) 2 (D) \(\sqrt{3}\)
Correct Option: (B)
8. If \(\tan A = \sqrt{3}\), then the value of \(\sec A\) is:
(A) 2 (B) \(\frac{\sqrt{3}}{2}\) (C) \(\frac{1}{2}\) (D) \(\frac{1}{\sqrt{3}}\)
Correct Option: (A)
9. The value of \(\tan 45^\circ\) is:
(A) 0 (B) 1 (C) \(\sqrt{3}\) (D) \(1/\sqrt{3}\)
Correct Option: (B)
10. If \(\cos 9\alpha = \sin \alpha\) and \(9\alpha < 90^\circ\), then the value of \(\tan 5\alpha\) is:
(A) \(\frac{1}{\sqrt{3}}\) (B) \(\sqrt{3}\) (C) 1 (D) 0
Correct Option: (C)
