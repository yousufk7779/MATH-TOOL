Chapter 4: Quadratic Equations

Class 10 - NCERT

1. Introduction

A quadratic equation is a polynomial equation of degree 2. In this chapter, we will learn how to solve quadratic equations using different methods - factorisation, completing the square, and quadratic formula. We will also learn to determine the nature of roots using the discriminant. Quadratic equations have wide applications in real-life problems involving area, speed, age, and many more scenarios.

2. Definitions
  • Quadratic Equation: An equation of the form ax² + bx + c = 0, where a, b, c are real numbers and a ≠ 0.
  • Standard Form: ax² + bx + c = 0 (terms arranged in descending order of degree).
  • Roots/Zeroes: Values of x that satisfy the quadratic equation.
  • Discriminant (D): The expression b² - 4ac that determines the nature of roots.
  • Factorisation: Expressing the quadratic as a product of two linear factors.
3. Key Points
  • A quadratic equation can have at most 2 roots.
  • If a = 0, the equation becomes linear, not quadratic.
  • The sum of roots = −ba and product of roots = ca.
  • For real roots, the discriminant must be ≥ 0.
  • When D > 0, roots are distinct; when D = 0, roots are equal; when D < 0, no real roots exist.
  • Always verify answers by substituting back into the original equation.
4. Formulas Used in This Chapter
Standard Form ax² + bx + c = 0 (a ≠ 0)
Quadratic Formula x = (-b ± √(b² - 4ac)) / 2a
Discriminant D = b² - 4ac
Sum of Roots α + β = −ba
Product of Roots αβ = ca
Pythagoras Theorem Hypotenuse² = Base² + Altitude²
(a + b)² a² + 2ab + b²
(a - b)² a² - 2ab + b²
(a + b)³ a³ + 3a²b + 3ab² + b³
5. Crux of the Chapter

Step 1: Convert any given equation to standard form ax² + bx + c = 0.

Step 2: Choose the appropriate method:

  • Factorisation: When factors are easy to find (integers or simple surds).
  • Quadratic Formula: When factorisation is difficult or for general cases.

Step 3: Check the discriminant to know the nature of roots before solving.

Step 4: In word problems, reject negative/invalid values based on context.

Step 5: Always verify your answer by substitution.

EXERCISE 4.1
Q.1(i) Check whether the following is a quadratic equation: (x + 1)² = 2(x - 3)
Solution:
LHS = (x + 1)² = x² + 2x + 1 (a + b)² = a² + 2ab + b²
RHS = 2(x - 3) = 2x - 6
Equation becomes: x² + 2x + 1 = 2x - 6
x² + 2x + 1 - 2x + 6 = 0
x² + 7 = 0
This is of the form ax² + bx + c = 0, where a = 1, b = 0, c = 7 Standard Form
Yes, it is a quadratic equation.
Q.1(ii) x² - 2x = (-2)(3 - x)
Solution:
RHS = (-2)(3 - x) = -6 + 2x
Equation: x² - 2x = -6 + 2x
x² - 2x - 2x + 6 = 0
x² - 4x + 6 = 0 Standard Form
Here a = 1, b = -4, c = 6 and a ≠ 0
Yes, it is a quadratic equation.
Q.1(iii) (x - 2)(x + 1) = (x - 1)(x + 3)
Solution:
LHS = (x - 2)(x + 1) = x² + x - 2x - 2 = x² - x - 2 Distributive Law
RHS = (x - 1)(x + 3) = x² + 3x - x - 3 = x² + 2x - 3 Distributive Law
Equation: x² - x - 2 = x² + 2x - 3
x² - x - 2 - x² - 2x + 3 = 0
-3x + 1 = 0
This is linear (no x² term, coefficient of x² = 0).
No, it is NOT a quadratic equation.
Q.1(iv) (x - 3)(2x + 1) = x(x + 5)
Solution:
LHS = (x - 3)(2x + 1) = 2x² + x - 6x - 3 = 2x² - 5x - 3 Distributive Law
RHS = x(x + 5) = x² + 5x
Equation: 2x² - 5x - 3 = x² + 5x
2x² - 5x - 3 - x² - 5x = 0
x² - 10x - 3 = 0 Standard Form
Yes, it is a quadratic equation.
Q.1(v) (2x - 1)(x - 3) = (x + 5)(x - 1)
Solution:
LHS = (2x - 1)(x - 3) = 2x² - 6x - x + 3 = 2x² - 7x + 3 Distributive Law
RHS = (x + 5)(x - 1) = x² - x + 5x - 5 = x² + 4x - 5
Equation: 2x² - 7x + 3 = x² + 4x - 5
2x² - 7x + 3 - x² - 4x + 5 = 0
x² - 11x + 8 = 0 Standard Form
Yes, it is a quadratic equation.
Q.1(vi) x² + 3x + 1 = (x - 2)²
Solution:
RHS = (x - 2)² = x² - 4x + 4 (a - b)² = a² - 2ab + b²
Equation: x² + 3x + 1 = x² - 4x + 4
x² + 3x + 1 - x² + 4x - 4 = 0
7x - 3 = 0
This is linear (no x² term).
No, it is NOT a quadratic equation.
Q.1(vii) (x + 2)³ = 2x(x² - 1)
Solution:
LHS = (x + 2)³ = x³ + 3x²(2) + 3x(4) + 8 = x³ + 6x² + 12x + 8 (a + b)³ identity
RHS = 2x(x² - 1) = 2x³ - 2x
Equation: x³ + 6x² + 12x + 8 = 2x³ - 2x
x³ + 6x² + 12x + 8 - 2x³ + 2x = 0
-x³ + 6x² + 14x + 8 = 0
This is cubic (highest power is 3).
No, it is NOT a quadratic equation.
Q.1(viii) x³ - 4x² - x + 1 = (x - 2)³
Solution:
RHS = (x - 2)³ = x³ - 3x²(2) + 3x(4) - 8 = x³ - 6x² + 12x - 8 (a - b)³ identity
Equation: x³ - 4x² - x + 1 = x³ - 6x² + 12x - 8
x³ - 4x² - x + 1 - x³ + 6x² - 12x + 8 = 0
2x² - 13x + 9 = 0 Standard Form
Here a = 2, b = -13, c = 9 (a ≠ 0)
Yes, it is a quadratic equation.
Q.2(i) The area of a rectangular plot is 528 m². The length of the plot (in metres) is one more than twice its breadth. Find the length and breadth of the plot.
Solution:
Let breadth = x metres
Then length = (2x + 1) metres
Area of rectangle = length × breadth Area = l × b
528 = (2x + 1) × x
528 = 2x² + x
2x² + x - 528 = 0 Standard Form
Required quadratic equation: 2x² + x - 528 = 0
Q.2(ii) The product of two consecutive positive integers is 306. Find the integers.
Solution:
Let first positive integer = x
Next consecutive integer = x + 1
Product = 306
x(x + 1) = 306
x² + x = 306
x² + x - 306 = 0 Standard Form
Required quadratic equation: x² + x - 306 = 0
Q.2(iii) Rohan's mother is 26 years older than him. The product of their ages (in years) 3 years from now will be 360. Find Rohan's present age.
Solution:
Let Rohan's present age = x years
Mother's present age = (x + 26) years
After 3 years:
Rohan's age = (x + 3) years
Mother's age = (x + 26 + 3) = (x + 29) years
Product of ages after 3 years = 360
(x + 3)(x + 29) = 360
x² + 29x + 3x + 87 = 360 Distributive Law
x² + 32x + 87 - 360 = 0
x² + 32x - 273 = 0 Standard Form
Required quadratic equation: x² + 32x - 273 = 0
Q.2(iv) A train travels a distance of 480 km at a uniform speed. If the speed had been 8 km/h less, then it would have taken 3 hours more to cover the same distance. Find the speed of the train.
Solution:
Let original speed = x km/h
Time taken = Distance/Speed = 480/x hours Time = Distance/Speed
New speed = (x - 8) km/h
New time = 480/(x - 8) hours
According to question: New time = Original time + 3
480/(x - 8) = 480/x + 3
Multiply by x(x - 8):
480x = 480(x - 8) + 3x(x - 8)
480x = 480x - 3840 + 3x² - 24x
0 = -3840 + 3x² - 24x
3x² - 24x - 3840 = 0
Divide by 3: x² - 8x - 1280 = 0 Standard Form
Required quadratic equation: x² - 8x - 1280 = 0

Q.1(i) Find the roots by factorisation: x² - 3x - 10 = 0
Solution:
We need two numbers whose product = -10 and sum = -3
Splitting Middle Term Method
Numbers are: -5 and +2 (because -5 × 2 = -10 and -5 + 2 = -3)
x² - 5x + 2x - 10 = 0
x(x - 5) + 2(x - 5) = 0 Factorisation by Grouping
(x - 5)(x + 2) = 0
Either x - 5 = 0 or x + 2 = 0 Zero Product Rule
x = 5 or x = -2
Roots are: x = 5 and x = -2
Q.1(ii) 2x² + x - 6 = 0
Solution:
Product = 2 × (-6) = -12, Sum = 1
Splitting Middle Term Method
Numbers: 4 and -3 (because 4 × -3 = -12 and 4 + (-3) = 1)
2x² + 4x - 3x - 6 = 0
2x(x + 2) - 3(x + 2) = 0 Factorisation by Grouping
(x + 2)(2x - 3) = 0
x + 2 = 0 or 2x - 3 = 0 Zero Product Rule
x = -2 or x = 32
Roots are: x = -2 and x = 32
Q.1(iii) √2 x² + 7x + 5√2 = 0
Solution:
Product = √2 × 5√2 = 5 × 2 = 10, Sum = 7

√2 x + 5 = 0 or x + √2 = 0 Zero Product Rule
x = -5/√2 = -5√2/2 or x = -√2
Roots are: x = -5√2/2 and x = -√2
Q.1(iv) 2x² - x + 1/8 = 0
Solution:

16x² - 8x + 1 = 0

Q.5 The altitude of a right triangle is 7 cm less than its base. If the hypotenuse is 13 cm, find the other two sides.
Solution:
Let base = x cm
Altitude = (x - 7) cm
Hypotenuse = 13 cm
By Pythagoras theorem:
Hypotenuse² = Base² + Altitude²
13² = x² + (x - 7)²
169 = x² + x² - 14x + 49 (a - b)² = a² - 2ab + b²
169 = 2x² - 14x + 49
2x² - 14x + 49 - 169 = 0
2x² - 14x - 120 = 0
x² - 7x - 60 = 0 (dividing by 2) Standard Form
Product = -60, Sum = -7
Splitting Middle Term Method
Numbers: -12 and 5
(x - 12)(x + 5) = 0 Zero Product Rule
x = 12 or x = -5 (rejected, length cannot be negative)
Base = 12 cm, Altitude = 12 - 7 = 5 cm
Base = 12 cm, Altitude = 5 cm
Q.6 A cottage industry produces a certain number of pottery articles in a day. It was observed on a particular day that the cost of production of each article (in rupees) was 3 more than twice the number of articles produced on that day. If the total cost of production on that day was Rs 90, find the number of articles produced and the cost of each article.
Solution:
Let number of articles = x
Cost of each article = (2x + 3) rupees
Total cost = Number × Cost per article
90 = x(2x + 3)
90 = 2x² + 3x
2x² + 3x - 90 = 0 Standard Form
Product = -180, Sum = 3
Splitting Middle Term Method
Numbers: 15 and -12
2x² + 15x - 12x - 90 = 0
x(2x + 15) - 6(2x + 15) = 0
(2x + 15)(x - 6) = 0 Zero Product Rule
x = -15/2 (rejected) or x = 6
Cost = 2(6) + 3 = 15 rupees
Number of articles = 6, Cost of each = Rs 15
