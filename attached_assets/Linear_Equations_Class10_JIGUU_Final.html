Chapter 3: Pair of Linear Equations in Two Variables

Class 10 - NCERT

1. Introduction

A linear equation in two variables represents a straight line when plotted on a graph. In this chapter, we will learn how to solve a pair of such equations using graphical and algebraic methods. These equations help us solve real-life problems involving two unknown quantities like cost-price, age, speed-distance, etc.

2. Definitions
  • Linear Equation: An equation of the form ax + by + c = 0, where a, b, c are real numbers and a, b are not both zero.
  • Pair of Linear Equations: Two linear equations in same two variables x and y.
  • Solution: An ordered pair (x, y) that satisfies both equations simultaneously.
  • Consistent System: A pair of equations that has at least one solution.
  • Inconsistent System: A pair of equations that has no solution (parallel lines).
  • Dependent System: A pair of equations that has infinitely many solutions (coincident lines).
3. Key Points
  • A pair of linear equations can have: unique solution, no solution, or infinitely many solutions.
  • If lines intersect → unique solution (consistent).
  • If lines are parallel → no solution (inconsistent).
  • If lines coincide → infinitely many solutions (dependent & consistent).
  • Substitution method: Express one variable in terms of other, then substitute.
  • Elimination method: Make coefficients equal, then add/subtract to eliminate one variable.
  • Cross-multiplication is a quick formula-based method for solving.
4. Formulas Used in This Chapter
General Form a₁x + b₁y + c₁ = 0, a₂x + b₂y + c₂ = 0
Unique Solution (Intersecting) a₁a₂ ≠ b₁b₂
No Solution (Parallel) a₁a₂ = b₁b₂ ≠ c₁c₂
Infinite Solutions (Coincident) a₁a₂ = b₁b₂ = c₁c₂
Cross-Multiplication xb₁c₂ − b₂c₁ = yc₁a₂ − c₂a₁ = 1a₁b₂ − a₂b₁
5. Crux of the Chapter

Step 1: First check the type of solution using ratios a₁a₂, b₁b₂, c₁c₂.

Step 2: Choose appropriate method:

  • Graphical: Plot both lines and find intersection point.
  • Substitution: When one equation is simple (like y = 2x).
  • Elimination: When coefficients are easy to match.

Step 3: Solve and find values of x and y.

Step 4: Verify by substituting back in both original equations.

Q.5 Half the perimeter of a rectangular garden, whose length is 4 m more than its width, is 36 m. Find the dimensions of the garden.
Solution:
Let length = x m, width = y m
Given that Length = Width + 4
∴ x = y + 4
Or x − y = 4 ... (i)
Also half perimeter = 36
∴ 2(Length + Breadth)2 = 36
x + y = 36 ... (ii)
Adding (i) and (ii), we have
2x = 40
x = 20, put x = 20 in (ii)
y = 36 − 20
y = 16
Hence, Length = 20 m, Width = 16 m
Q.6 Given the linear equation 2x + 3y − 8 = 0, write another linear equation in two variables such that the geometrical representation of the pair so formed is:
Solution:
(i) Intersecting lines:
For intersection: a₁a₂ ≠ b₁b₂
One example: 3x + 2y − 7 = 0 23 ≠ 32
(ii) Parallel lines:
For parallel: a₁a₂ = b₁b₂ ≠ c₁c₂
One example: 2x + 3y − 12 = 0 22 = 33 ≠ 812
(iii) Coincident lines:
For coincident: a₁a₂ = b₁b₂ = c₁c₂
One example: 4x + 6y − 16 = 0 24 = 36 = 816
(i) 3x + 2y − 7 = 0, (ii) 2x + 3y − 12 = 0, (iii) 4x + 6y − 16 = 0

x = 14 − y ... (iii)

x = 14 − 5 = 9
x = 9

Adding: 2x = 40
x = 20
From (i): y = 36 − 20 = 16
Length = 20 m, Width = 16 m
Example 3 Solve using substitution: 7x − 15y = 2, x + 2y = 3
Solution:
The given equations are:
7x − 15y = 2 ... (i)
x + 2y = 3 ... (ii)
From (ii): x = 3 − 2y ... (iii)
Substitute in (i): 7(3 − 2y) − 15y = 2
21 − 14y − 15y = 2
−29y = −19
y = 1929
x = 3 − 2 × 1929 = 87 − 3829 = 4929
x = 4929, y = 1929

Chapter Summary
  • A pair of linear equations can have: unique solution, no solution, or infinitely many solutions.
  • Graphical Method: Plot both lines; intersection point is the solution.
  • Substitution Method: Express one variable in terms of other, then substitute.
  • Elimination Method: Make coefficients equal, then add/subtract to eliminate one variable.
  • Condition Check: Compare a₁a₂, b₁b₂, c₁c₂ to determine type of solution.
  • For a₁a₂ ≠ b₁b₂ → Unique solution (lines intersect).
  • For a₁a₂ = b₁b₂ = c₁c₂ → Infinite solutions (coincident lines).
  • For a₁a₂ = b₁b₂ ≠ c₁c₂ → No solution (parallel lines).
